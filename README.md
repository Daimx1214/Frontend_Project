# Frontend Practice

A collection of my hands-on HTML, CSS, and JavaScript practice files — built while learning frontend development from scratch.

---

## What's Inside

### HTML
- Semantic structure and layout basics
- Forms, tables, and media elements
- Building real page structures

### CSS
- Selectors, specificity, and the box model
- Flexbox and Grid layouts
- Positioning, floats, and display modes
- Animations and transforms
- Media queries and responsive design
- Backgrounds and sizing

### JavaScript
- DOM manipulation
- Events and interactivity
- Logic and problem-solving exercises
